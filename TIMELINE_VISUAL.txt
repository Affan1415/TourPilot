TECHNOLOGY STACK:
  • Database:     Supabase (PostgreSQL) with RLS policies
  • Backend:      Next.js 14 App Router + API Routes
  • Frontend:     React 19 + shadcn/ui components
  • Styling:      Tailwind CSS 4.0 (with dark mode)
  • Payments:     Stripe (webhooks, 3D Secure)
  • Emails:       Resend API (campaigns, notifications)
  • Offline:      IndexedDB + Service Workers
  • Testing:      Jest, React Testing Library, E2E tests
